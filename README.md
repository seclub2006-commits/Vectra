# Vectra – Универсальная торговая платформа

## Возможности

- **Автоматические торговые стратегии**: EMA, RSI, Grid, и другие – легко добавлять через наследование `BaseBot`.
- **Ручной терминал** с продвинутым графиком: рисование линий тренда, уровней, текстовых меток; отображение открытых позиций и TP/SL.
- **Поддержка нескольких источников рыночных данных**: WebSocket, REST polling, кэш в SQLite, воспроизведение из CSV.
- **Эмуляция торговли** – безопасное тестирование стратегий без реальных средств.
- **Гибкое управление коннекторами** к биржам (Bitget, легко добавлять другие).
- **Асинхронное ядро на gRPC** – отдельный процесс, управление ботами через сеть.
- **GUI на PyQt5** – современный тёмный интерфейс, графики на pyqtgraph.
